# go-tmp-crypt

---

Golang web application to store encrypted files with an expiration date

## Docker

```bash
$ git clone https://github.com/smolveau/go-tmp-crypt.git
$ cd go-tmp-crypt
$ docker-compose down
$ docker-compose up -d
```

Accessible on localhost:9090

### Stack

* Go(lang)
* Goland IDE by JetBrains
* PostgreSQL
* Docker
